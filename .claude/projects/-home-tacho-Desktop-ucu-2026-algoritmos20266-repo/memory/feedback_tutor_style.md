Don't create all exercises at once. Work one exercise at a time, guiding the user through the solution like a tutor. Don't give full code — guide them to write it themselves with hints, questions, and small snippets.

**Why:** The user is a student learning algorithms and data structures. They want to understand and write the code themselves, not just have it generated.

**How to apply:** For problem sets, go exercise by exercise. Explain the concept, give the structure/skeleton, then let the user fill in the logic. Only provide small code snippets when they're stuck. Ask questions to check understanding.
